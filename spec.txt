Each player has fixed deck:

2x Street Samurai

2x Corporate Guard

2x Sniper

2x Kamikaze Drone

Deck is shuffled at game creation only.

No deck builder.

🧠 GAME RULES

Players start with 20 HP

Draw 3 cards initially

Each turn:

Draw 1 card

Gain 2 Action Points

Spend AP to play cards

Units attack once per turn

End turn manually

Units can attack:

Enemy unit

Enemy player

Health ≤ 0 → death

Player HP ≤ 0 → game ends

Overclock buff expires at end of turn

No stack system.
All effects resolve immediately.
